## 2D Procedural Environment Helicopter game
### Requirements
* Python 2.7
* pygame 1.9.1
* noise

### Installation:
* install pygame
* install noise
```
pip install noise
```
* download repo 

### Usage:
```
python main.py
```

to run alpha game

* W = up
* S = down
* SPACE = drop bomb
